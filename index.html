API Information

This API retrieves the information of a given pokemon such as their type, move sets, encounters, evolutions, items, and much more. This specific impementation of the API retrieves and displays the name, type, and description of the pokemon.

Where this API was found: API Documentation

Options used for the API request:
- used GET with the following URL to get the pokemon name and type: http://pokeapi.co/api/v2/pokemon/{name}
- used GET with the following URL to get the pokemon species description: http://pokeapi.co/api/v2/pokemon-species/{name}

Application for this API:
1. To be used for a large Pokemon directory or gallery
2. To gather information for a Wikipedia of Pokemon, an online handbook, or a game
